# MyCnblogsTheme
我的博客园主题美化工程仓库

> 每个主题下面有两个文件夹，一个为所搬运博主的源工程，一个是我根据博主的教程，按照我的喜好修改过的工程MyThemeProject，在我的工程下面又有两个文件夹，ScriptFile为源工程博主所提供的脚本文件，需要上传至博客园；BeautifyFile为美化文件，即侧边栏代码、页面定制CSS代码、页首代码、页脚代码。
